# Agent Signal Bar v1.3.0

## English

### Highlights

- Added a floating signal light that can stay visible on the desktop while keeping the existing menu bar signal.
- Added desktop resizing, drag positioning, vertical/horizontal floating layouts, and persisted floating-light placement.
- Added separate completion, green-flash, and alert sound choices, including the New Zealand crossing sounds and bundled generated tones.
- Reorganized settings so general signal toggles stay in General while detailed status bar, floating light, sound, and effect controls live in their own sections.

### Changes

- The floating light now follows the same live signal snapshot and animation clock as the menu bar light.
- The floating light shows a compact running-agent badge only while monitored agents are active.
- The floating light information popover now focuses on currently running sessions and stays hidden when idle or monitoring is paused.
- New installs default to vertical floating light layout, simple status bar menu, launch at login enabled, standard breathing strength, and automatic update checks enabled.
- The packaged app now includes the new floating-light audio resources.

### Fixes

- Fixed local integration verification so it uses the full Xcode toolchain when available instead of failing under Command Line Tools-only Swift.
- Fixed login-item plist generation in the installer so paths with XML-special characters are written safely.
- Fixed several floating-light resize, cursor, badge, and popover placement issues found during local testing.

- `DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift test`
- `./script/verify_local_integrations.sh`

## 简体中文

### 亮点

- 新增桌面悬浮信号灯，可以在保留原状态栏信号灯的同时常驻桌面。
- 新增桌面缩放、拖动定位、横向/竖向悬浮布局，以及悬浮灯位置持久化。
- 新增完成音、绿灯闪烁音、告警音的独立选择，包含新西兰红绿灯声音和内置生成提示音。
- 重新整理设置：通用里保留基础开关，状态栏、悬浮灯、声音和灯效细项放到各自区域。

### 改进

- 悬浮灯现在和状态栏信号灯使用同一份实时状态快照和动画时钟。
- 只有监控到 Agent 正在运行时，悬浮灯才显示紧凑的运行数量角标。
- 悬浮灯信息浮层现在聚焦显示正在运行的会话；空闲或暂停监控时不会显示。
- 新安装默认使用竖向悬浮灯、简约状态栏菜单、开机自启动开启、标准呼吸强度、自动检查更新开启。
- 打包后的 App 现在会包含新的悬浮灯声音资源。

### 修复

- 修复本地集成验证脚本：可用完整 Xcode 时会自动使用 Xcode 工具链，避免 Command Line Tools 的 Swift 崩溃。
- 修复安装脚本登录项 plist 生成逻辑：安装路径包含 XML 特殊字符时也能安全写入。
- 修复本地测试中发现的多处悬浮灯缩放、光标、角标和信息浮层位置问题。
