# Agent Signal Bar v1.3.1

## English

### Highlights

- Patch release for the v1.3 floating signal light.
- Fixes a crash that could happen after floating-light sounds played repeatedly.

### Changes

- The Done Effect menu in Advanced settings now opens upward so all options remain visible near the bottom of the window.
- README release badges now point at v1.3.1.

### Fixes

- Fixed the floating signal sound player lifetime. Each `AVAudioPlayer` is now retained until its finish or decode-error callback completes, with a delayed release fallback, so green-flash and completion sounds cannot call back into a released player.

### Release verification

- `git diff --check`
- `DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift build`
- `DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer swift test`
- `./script/build_and_run.sh --verify`
- `DEVELOPER_DIR=/Applications/Xcode.app/Contents/Developer ./script/verify_release_all.sh --skip-package`
- Local smoke test: emitted a temporary `working` signal for 12 seconds to exercise the green-flash sound path, then reset the temporary session.

## 简体中文

### 亮点

- v1.3 悬浮信号灯的补丁版本。
- 修复悬浮灯声音连续播放后可能触发的崩溃。

### 改进

- 高级设置里的“完成灯效”菜单现在会向上展开，靠近窗口底部时也能看到完整选项。
- README 版本徽章已更新到 v1.3.1。

### 修复

- 修复悬浮灯声音播放器生命周期问题。每个 `AVAudioPlayer` 现在会保留到播放完成或解码错误回调结束，并带有延迟释放兜底，避免绿灯闪烁音和完成音回调到已经释放的播放器。

- 本地烟测：临时发送 `working` 状态 12 秒，覆盖绿灯闪烁音路径，然后重置临时会话。
